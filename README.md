# Czytaj.pl — Kreator Bohatera (Avatar Builder)

A small avatar builder for Czytaj.pl. Pick hair, blouse, pants and an optional
companion animal (bird or dog), then download a transparent PNG of your hero.

- **Windows 98 pixel theme** UI
- 18 pre-rendered character combinations (hair × blouse × pants)
- Bird sits on the shoulder, dog stands beside the girl
- "Pobierz PNG" composites character + animal into one transparent PNG (canvas)

## Run
Open `index.html` in a browser, or serve the folder and visit it.
Hosted via GitHub Pages.

## Structure
- `index.html` — the whole app (HTML/CSS/JS inline)
- `assets/` — 18 character PNGs (`{hair}_{blouse}_{pants}.png`), `animal_bird.png`,
  `animal_dog.png`, `shadow.png`, `download-button.svg`
